This release introduces bundled services that run alongside projects, real-time certificate verification, and major improvements to resource management and UI.

- **Bundled Services**: Per-project databases (MySQL, PostgreSQL, MongoDB) and admin tools that run alongside specific projects
- **Credentials Viewer**: Copy-to-clipboard functionality for bundled service credentials
- **Database Operations**: Dump, restore, and copy data between bundled services
- **Real-time Caddy Verification**: Ensures HTTPS certificates are properly configured
- **Enhanced Resources UI**: Redesigned table with grouping and improved pagination
